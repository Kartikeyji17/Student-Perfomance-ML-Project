# 🎓 Student Performance Prediction (Machine Learning)

A full-stack machine learning project that predicts student math scores based on socio-economic and academic factors using an end-to-end ML pipeline and web deployment.

---

## 🚀 Overview

This project builds a predictive model to estimate student performance using features like gender, parental education, lunch type, and test preparation.

It demonstrates:

* Complete ML lifecycle (data → training → deployment)
* Feature engineering and preprocessing
* Model evaluation and optimization
* Web-based user interaction

---

## 🧠 Problem Statement

Educational institutions often lack early indicators of student performance.

👉 This project solves that by:

* Predicting student scores in advance
* Identifying key influencing factors
* Enabling data-driven academic support

---

## 🛠️ Tech Stack

* **Languages:** Python
* **ML Libraries:** scikit-learn, Pandas, NumPy
* **Model Used:** Random Forest / Decision Tree
* **Backend:** Flask
* **Frontend:** HTML, CSS
* **Tools:** Jupyter Notebook

---

## 📊 Features

* Predicts student math score based on input parameters
* Interactive web interface
* Data preprocessing pipeline
* Model training & evaluation system
* Modular project structure

---

## 📈 Model Performance

* Achieved **~85% accuracy** on test data
* Evaluated using:

  * Mean Absolute Error (MAE)
  * R² Score
